TITRE : Infolingo
AUTEURS : Yevhenii Cherevko, Gwenael Marguet
DESCRIPTION : Ce site a été developpé aux fins éducatifs pour apprendre les bases de html/css et du framework Flask.
              Le but du site est d'enséigner de différentes langues de programmation telles que Python, C, Javascript.
              Le site contient un forum où les utilisateurs inscrits peuvent poser leurs questions concernant les langues de programmation, y répondre et les liker ou disliker.
              Tout de même le site propose un exercise au hasard pour les utilisateurs inscrits.
PROCEDURE A SUIVRE POUR EXECUTER LE PROGRAMME : 1. Ouvrir le dossier contenant la racine du projet.
                                                2. Taper dans le terminal : « python3 server.py ».
