TITRE : Infolingo
AUTEURS : Yevhenii Cherevko, Gwenael Marguet
DESCRIPTION : Ce site a été développé à des fins éducatives pour apprendre les bases du HTML/CSS et du framework Flask.
              Le but du site est d’enseigner différentes langues de programmation telles que Python, C et JavaScript.
              Le site contient un forum où les utilisateurs inscrits peuvent poser leurs questions concernant les langages de programmation, y répondre, ainsi que liker ou disliker les réponses.
              Le site propose également un exercice au hasard pour les utilisateurs inscrits.
PROCEDURE A SUIVRE POUR EXECUTER LE PROGRAMME : 1. Ouvrir le dossier contenant la racine du projet.
                                                2. Créer l'environnement Venv à l'aide du fichier "requirements.txt" 
                                                3. Taper dans le terminal : «.venv/bin/python server.py».
