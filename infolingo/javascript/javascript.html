Le langage JavaScript

JavaScript est un langage de programmation de haut niveau principalement utilisé pour rendre les pages web interactives. Créé en 1995, il est devenu l'un des piliers du développement web, avec HTML et CSS. JavaScript permet de manipuler les éléments d'une page web, de gérer les événements utilisateurs et de créer des applications web dynamiques. Cette introduction vous permettra de comprendre les bases de JavaScript et d'implémenter des fonctionnalités simples sur une page web.

Chapitre 1 : JavaScript "Hello, World!"

Comment fonctionne le programme "Hello, World!" en JavaScript ?

  1. Le code JavaScript peut être intégré directement dans une page HTML à l'intérieur de balises <script>.

  2. La fonction console.log() est utilisée pour afficher des messages dans la console du navigateur.

  3. Pour afficher un message à l'utilisateur, vous pouvez utiliser la fonction alert(), qui affiche une boîte de dialogue.

<script>

 alert("Hello, World!");

 console.log("Hello, World!");

</script>

Chapitre 2 : JavaScript Variables

Déclaration de variables en JavaScript

  1. En JavaScript, vous pouvez déclarer des variables avec let, const ou var.

  2. let est utilisé pour déclarer des variables qui peuvent changer de valeur.

  3. const est utilisé pour déclarer des constantes, c'est-à-dire des variables dont la valeur ne changera jamais.

  4. var est une ancienne façon de déclarer des variables, mais elle est moins utilisée aujourd'hui car elle peut causer des comportements imprévisibles.

<script>

 let name = "Alice";

 const age = 25;

 console.log("Nom : " + name);

 console.log("Âge : " + age);

</script>

Chapitre 3 : JavaScript Conditions

Utilisation des conditions en JavaScript

  1. Les conditions permettent de tester des expressions et d'exécuter du code en fonction du résultat.

  2. En JavaScript, les conditions sont définies avec if, else if et else.

<script>

 let age = 18;

 if (age >= 18) {

  console.log("Vous êtes majeur");

 } else {

  console.log("Vous êtes mineur");

 }

</script>

Sortie :

Si l'âge est 18 ou plus, "Vous êtes majeur" s'affiche dans la console.
