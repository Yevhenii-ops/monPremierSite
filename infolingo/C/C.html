Le langage C

C est un langage de programmation impératif, généraliste et de bas niveau. Inventé au début des années 1970 pour réécrire Unix, le langage C est encore largement utilisé. De nombreux langages plus modernes comme C++, C#, Java et PHP ou JavaScript ont repris une syntaxe similaire au C et reprennent en partie sa logique. C offre au développeur une marge de contrôle importante sur la machine (notamment sur la gestion de la mémoire) et est de ce fait utilisé pour réaliser les « fondations » (compilateurs, interpréteurs, …) de ces langages plus modernes. Ces cours vous permettent d'obtenir la base sur le langage C pour vous lancer dans ce domaine de travail.

Chapitre 1 : C "Hello, World!" Programme

Comment le programme "Hello, World!" fonctionne ?

  1. Le #include est une commande de préprocesseur qui indique au compilateur d'inclure le contenu du fichier stdio.h (entrée et sortie standard) dans le programme

  2. The stdio.h file contains functions such as scanf() and printf() to take input and display output respectively.

  3. Si vous utilisez la fonction printf() sans écrire #include , le programme ne compilera pas.

  4. The execution of a C program starts from the main() function.

  5. printf() is a library function to send formatted output to the screen. In this program, printf() displays Hello, World! text on the screen.

  6. The return 0; statement is the "Exit status" of the program. In simple terms, the program ends with this statement.

#include "stdio.h"

int main() {

 // printf() affiche la chaîne à l'intérieur du guillemet

 printf("Hello, world!");

return 0;

}

Chapitre 2 : C Programme pour afficher un nombre entré par l'utilisateur

#include "stdio.h"

int main() {

 int number

 printf("Enter an integer: ");

 // lit et stocke l'entrée

 scanf("%d", &number);

 print("You entered: %d", number);

return 0;

}

Output :

Enter an integer : 25

You entered : 25

  1. Dans le programme, variable integer (nombre entière) est déclaré

    int number;

  2. Ensuite, l'utilisateur est invité à saisir un nombre entière. Ce nombre est stocké dans la variable number.

    printf("Enter the integer: ");

    scanf("%d", number);

  3. Enfin, la valeur stockée dans number est affichée à l'écran à l'aide de printf().
