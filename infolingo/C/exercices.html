Exercices de programmation en C

Ces exercices permettent de pratiquer les bases de la programmation en C.

Exercice 1 : Afficher le carré d'un nombre

Consigne : Écrire un programme en C qui demande à l'utilisateur d'entrer un nombre entier, puis affiche le carré de ce nombre.

#include <stdio.h>

int main() {

 int number, square;

 printf("Entrez un nombre : ");

 scanf("%d", &number);

 square = number * number;

 printf("Le carré de %d est : %d", number, square);

return 0;

}

Sortie :

Entrez un nombre : 4

Le carré de 4 est : 16

Exercice 2 : Vérifier si un nombre est pair ou impair

Consigne : Écrire un programme en C qui détermine si un nombre entré par l'utilisateur est pair ou impair.

#include <stdio.h>

int main() {

 int number;

 printf("Entrez un nombre : ");

 scanf("%d", &number);

 if (number % 2 == 0)

  printf("%d est pair", number);

 else

  printf("%d est impair", number);

return 0;

}

Sortie :

Entrez un nombre : 7

7 est impair

Exercice 3 : Calculer la somme de deux nombres

Consigne : Écrire un programme en C qui demande à l'utilisateur d'entrer deux nombres entiers, puis affiche leur somme.

#include <stdio.h>

int main() {

 int num1, num2, sum;

 printf("Entrez deux nombres : ");

 scanf("%d %d", &num1 , &num2);

 sum = num1 + num2;

 printf("La somme est : %d", sum);

return 0;

}

Sortie :

Entrez deux nombres : 3 5

La somme est : 8
