Chapitre 3 : Instructions conditionnelles if...else en Python

Les instructions conditionnelles permettent de prendre des décisions en fonction des conditions spécifiées. En Python, elles s'écrivent principalement avec les instructions if, elif, et else.

La syntaxe de base est la suivante :

if (condition):

 # instructions si la condition est vraie

elif (condition):

 # instructions si la première condition est fausse et celle-ci est vraie

else:

 # instructions si toutes les conditions précédentes sont fausses

Le bloc if s'exécute si la condition est vraie. Sinon, le code passe à l'instruction elif (si présente) ou else.

Conditions booléennes

Les conditions peuvent être basées sur des expressions qui se résolvent en True ou False. Par exemple :

x = 10

if (x > 5):

 print("x est supérieur à 5")

Syntaxe de l'instruction if

L'instruction if en Python permet d'exécuter un bloc de code seulement si une condition est vraie. Par exemple :

num = 3

if (num > 0):

 print("Nombre positif")

# Output: Nombre positif

Le code print("Nombre positif") est exécuté uniquement si num est supérieur à 0.


Instruction if...else

L'instruction if...else permet de définir un bloc de code à exécuter si la condition est fausse. Exemple :

num = -1

if (num >= 0):

 print("Nombre positif ou zéro")

else:

 print("Nombre négatif")

# Output: Nombre négatif

Dans cet exemple, si la valeur de num est inférieure à 0, le code du bloc else est exécuté.
