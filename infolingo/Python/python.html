Python

Python est un langage de programmation interprété, multiparadigme et multiplateformes. Il favorise la programmation impérative structurée, fonctionnelle et orientée objet. Il est doté d'un typage dynamique fort, d'une gestion automatique de la mémoire par ramasse-miettes et d'un système de gestion d'exceptions ; il est ainsi similaire à Perl, Ruby, Scheme, Smalltalk et Tcl.

Chapitre 1 : Votre premier programme Python

Le programme suivant affiche Hello, World ! sur l'écran.

print("Hello, World!")

Félicitations pour avoir écrit votre premier programme Python. Voyons maintenant comment fonctionne le programme ci-dessus.

  1. En Python, tout ce qui se trouve dans print est affiché à l'écran.

  2. Nous pouvons également utiliser des guillemets simples pour imprimer du texte à l'écran. Par exemple :

    3. print('Hello, World!')

    Pour être cohérent, nous utiliserons des guillemets doubles tout au long des tutoriaux.

Chapitre 2 : Variables et littéraux en Python

Variables en Python

  1. En programmation, une variable est un conteneur (zone de stockage) destiné à contenir des données. Par exemple :

    2. number = 10

    Ici, le nombre est une variable stockant la valeur 10.

  2. Comme nous pouvons le voir dans l'exemple ci-dessus, nous utilisons l'opérateur d'affectation = pour attribuer une valeur à une variable.

# assign value to site_name variable

 site_name = 'Infolingo'

 print(site_name);
